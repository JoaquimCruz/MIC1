# MIC-1

Repositório criado para hospedar o projeto MIC-1, da disciplina Arquitetura e Organização de Computadores I (AOCI). O MIC-1 representa todos os componentes que compõe a Unidade Lógica Aritmética (ULA - 32 bits) e todos os registradores que compõe o Banco de Registradores que é acoplado a ULA. Para tal representação, foi utilizado o software Quartus II - 13.1.0 Build 162 10/23/2013 SJ Web Edition. O projeto tem suporte para compilação nas famílias Cyclone III e Cyclone II.
